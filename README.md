## Commit Convention
**커밋은 작업 단위**
| 이름 | 내용 |
| ------ | ------ | 
|✨ Feat:   |새로운 기능에 대한 커밋 |
|🐛 Fix: | 버그 수정 |
|💄 Design: | CSS 등, UI 변경의 경우 |
|♻️ Refactor: | 코드 구조, 형식 개선 |
|💡 Comment: | 필요한 주석 추가, 변경 |
|🚚 Rename: | 파일 혹은 폴더명 수정 |
|🔥 Remove: | 파일 삭제 |
|🏷 Type: | 타입 추가 및 수정 |
|📝 Doc: | 문서 수정 |
